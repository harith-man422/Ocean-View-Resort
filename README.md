📖 Overview

This system is developed to simplify hotel operations by automating essential tasks such as customer check-in/check-out, billing, and room management. It provides an intuitive interface for staff and ensures efficient handling of hotel workflows.

✨ Features
User Authentication (Signup & Login)
Homepage Dashboard
Customer Check-In
Customer Check-Out
Billing System
Room Management
Help Section for New Users
Logout & Exit System
